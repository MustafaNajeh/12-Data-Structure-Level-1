# 📚 تصنيف هياكل البيانات (Classification of Data Structure)

---

# 🌟 ما هي أنواع الـ Data Structure؟

تنقسم الـ **Data Structure** إلى نوعين رئيسيين:

```
Data Structure
│
├── Primitive
│   ├── Integer
│   ├── Float
│   ├── Char
│   └── Pointer
│
└── Non-Primitive
    ├── Linear
    └── Non-Linear
```

---

# 🟢 أولاً: Primitive Data Structure

هي هياكل البيانات الأساسية.

✨ تتميز بأنها:

- تُعد من الأساسيات الموجودة داخل اللغة.
- يستطيع الحاسوب التعامل معها مباشرة.

### أمثلة:

- Integer
- Float
- Char
- Pointer

---

# 🔵 ثانياً: Non-Primitive Data Structure

هي هياكل بيانات أكثر تعقيدًا.

✨ تتميز بأنها:

- تُبنى باستخدام **Primitive Data Structure**.
- تساعد على تنظيم مجموعة من البيانات.
- قد تحتوي على بيانات من نفس النوع أو من أنواع مختلفة.
- عند تصميمها يجب مراعاة العمليات التي ستُجرى عليها.

---

# ⚖️ الفرق بين Primitive و Non-Primitive

## 🟢 Primitive

- هياكل أساسية.
- موجودة داخل لغة البرمجة.
- أمثلة:
  - Integer
  - Float

---

## 🔵 Non-Primitive

- تُبنى من عدة **Primitive Data Structure** مرتبطة معًا.

أمثلة:

- Array
- Linked List
- Binary Search Tree
- Tree
- Graph

---

# 📂 أنواع Non-Primitive Data Structure

تنقسم إلى قسمين:

```
Non-Primitive
│
├── Linear
│   ├── Array
│   ├── Linked List
│   ├── Stack
│   └── Queue
│
└── Non-Linear
    ├── Tree
    └── Graph
```

---

# 📏 Linear Data Structure

في هذا النوع تكون البيانات مرتبة بشكل متسلسل.

كل عنصر يكون مرتبطًا بالعنصر الذي قبله والذي بعده.

### أمثلة

- Array
- Stack
- Queue
- Linked List

---

# 🌳 Non-Linear Data Structure

في هذا النوع لا تكون البيانات مرتبة بشكل متسلسل.

ولا يمكن المرور على جميع العناصر في مرة واحدة فقط.

### أمثلة

- Tree
- Graph

---

# 📦 Static و Dynamic Data Structure

## 📌 Static Data Structure

✨ المميزات:

- حجم الذاكرة ثابت.
- الوصول إلى العناصر أسهل.

### مثال

- Array

---

## 🔄 Dynamic Data Structure

✨ المميزات:

- حجمها غير ثابت.
- يمكن أن يتغير أثناء تشغيل البرنامج.

### أمثلة

- Stack
- Queue

---

Data Structure

        └── Graph
```
